# 🏥 NovaMed Clinic Website

A modern, premium healthcare clinic website built using **HTML, CSS, and JavaScript**. The website features a dark-themed user interface, interactive animations, responsive design, appointment booking forms, doctor profiles, testimonials, FAQs, and an engaging healthcare-focused user experience.

## 🌐 Live Demo

https://clinic-website-omega-lemon.vercel.app/

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Aditya19012005/clinic_website.git
```

Navigate to the project folder:

```bash
cd clinic_website
```

Open the project:

```bash
index.html
```

or run using VS Code Live Server.
